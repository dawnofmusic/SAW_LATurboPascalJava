SAW Language Adapter TurboPascal Java
=====================================
About this project
------------------
Some tools created to simplify the process of migrating old TurboPascal code 
snippets into Java. Especially UI stuff. 

This project has been developed using the Eclipse IDE (http://www.eclipse.org).

Programming Language: Java (JavaSE-1.6)
